# 站点基本公约
```
1.讨论时注意文明用语
2.上传贡献软件时，注意留下您的顶顶网名，我们看到后会即时在下一期的视频测试使用和表彰
3.不要上传病毒，流氓软件等不好的东西，我们要的是优质开源软件
就这么多，不要问我为什么没有第四点！！！
```
